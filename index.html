Accessibility Tab Demo

Accessibility Tab

The accessibility tab is visible in the corner of the page. Click it to be redirected to the accessibility configuration page.

Default Implementation:

<eclec-tech enable-large-font />

Position Options

You can position the accessibility tab in different corners using the button-position attribute:

Top Left

<eclec-tech button-position="top-left" />

Top Right

<eclec-tech button-position="top-right" />

Bottom Left (Default)

<eclec-tech />

Bottom Right

<eclec-tech button-position="bottom-right" />

All Configuration Options

  • enable-large-font - Enable large font accessibility feature
  • config-url - Set custom URL for the accessibility configuration page (optional)
  • open-in-new-tab - Open configuration in a new tab instead of redirecting
  • button-position - Set the position of the accessibility tab (top-left, top-right, bottom-left, bottom-right)
